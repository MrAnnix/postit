# postit
Simple microblogging web app
## Prerequisites
- Oracle JDK or OpenJDK (version 8 or higher)
- MySQL database
- A compatible browser ;)

## Steps
```sh
git clone https://github.com/MrAnnix/postit.git # Or clone your own fork
cd postit
chmod +x gradlew
./gradlew bootRun
# Go to http://localhost:8080
```
